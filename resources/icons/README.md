# 图标文件说明

## 需要的图标文件

### 1. 应用程序图标 (app_icon.ico)

- **格式**: ICO
- **尺寸**: 256x256 (推荐包含多个尺寸: 16x16, 32x32, 48x48, 256x256)
- **用途**: 
  - 应用程序窗口图标
  - 任务栏图标
  - 打包后的 exe 文件图标

### 2. 系统托盘图标 (tray_icon.png)

- **格式**: PNG (支持透明背景)
- **尺寸**: 32x32 或 64x64
- **用途**: 系统托盘显示的图标

## 图标设计建议

### 设计风格
- 简洁现代
- 易于识别
- 适合深色和浅色背景

### 图标主题
建议使用以下元素：
- 亮度/对比度符号 (☀️ 🌙)
- 调节滑块图标
- 显示器图标
- Gamma 曲线图标

### 颜色方案
- 主色: #0d7377 (青色)
- 辅色: #14ffec (亮青色)
- 背景: 透明或白色

## 临时解决方案

在没有图标的情况下，程序仍然可以正常运行：
- 窗口会使用系统默认图标
- 托盘图标会显示为空白或默认图标

## 添加图标后

将图标文件放置在此目录下：
```
resources/icons/
├── app_icon.ico      # 应用程序图标
└── tray_icon.png     # 托盘图标
```

然后重新运行或打包程序即可。
