# GammaTool - 屏幕亮度调节工具

一个轻量级、功能强大的 Windows 屏幕亮度调节工具，支持灰度、亮度、对比度以及 RGB 三色通道的自定义调节。

## ✨ 主要特性

- 🎨 **多维度调节**: 支持亮度、对比度、灰度以及 RGB 三色通道独立调节
- ⌨️ **全局热键**: 自定义快捷键，随时随地快速调节
- 💾 **配置保存**: 自动保存用户设置，下次启动自动恢复
- 🔄 **一键恢复**: 快速恢复到系统默认设置
- 🎯 **系统托盘**: 最小化到系统托盘，不占用任务栏空间
- 🎭 **精美界面**: 现代化扁平设计，支持深色/浅色主题
- 📦 **体积小巧**: 打包后仅 15-20 MB，绿色免安装
- 🚀 **性能优异**: 低资源占用，响应迅速

## 📋 功能列表

### 核心功能

1. **亮度调节** (0-200%)
   - 默认值: 100%
   - 支持实时预览
   - 支持热键快速调节

2. **对比度调节** (0-200%)
   - 默认值: 100%
   - 增强画面层次感
   - 适合不同使用场景

3. **灰度调节** (0-100%)
   - 默认值: 0% (无灰度)
   - 护眼模式
   - 减少色彩刺激

4. **RGB 通道调节** (0-255)
   - 红色通道独立调节
   - 绿色通道独立调节
   - 蓝色通道独立调节
   - 实现色温调节效果

### 辅助功能

- ⌨️ **自定义热键**: 为常用操作设置快捷键
- 💾 **配置管理**: 自动保存和加载用户配置
- 🔄 **一键恢复**: 快速恢复默认设置
- 🎯 **系统托盘**: 后台运行，随时调用
- 🚀 **开机自启**: 可选的开机自动启动

## 🖥️ 系统要求

## 🎮 使用说明

### 基本操作

1. **启动程序**: 双击运行，主窗口会显示所有调节选项
2. **调节参数**: 拖动滑块或输入数值进行调节
3. **实时预览**: 调节会立即应用到屏幕
4. **保存设置**: 设置会自动保存，下次启动自动恢复
5. **最小化**: 点击最小化按钮，程序会隐藏到系统托盘

### 默认热键

| 热键 | 功能 |
|------|------|
| `Ctrl+Alt+Up` | 增加亮度 (+5%) |
| `Ctrl+Alt+Down` | 降低亮度 (-5%) |
| `Ctrl+Alt+R` | 恢复默认设置 |
| `Ctrl+Alt+G` | 显示/隐藏主窗口 |

*所有热键都可以在设置中自定义*

### 系统托盘菜单

- **显示主窗口**: 打开主界面
- **快速调节**: 快速选择预设值
- **恢复默认**: 一键恢复默认设置
- **开机自启**: 开启/关闭开机自动启动
- **退出程序**: 退出并恢复默认设置

## 🛠️ 开发指南

### 项目结构

```
GammaTool/
├── src/                    # 源代码
│   ├── main.py            # 程序入口
│   ├── gamma_engine.py    # Gamma 调节引擎
│   ├── main_window.py     # 主窗口界面
│   ├── hotkey_manager.py  # 热键管理器
│   ├── config_manager.py  # 配置管理器
│   ├── tray_icon.py       # 系统托盘
│   └── utils.py           # 工具函数
├── resources/             # 资源文件
│   ├── icons/            # 图标
│   └── styles/           # 样式表
├── config/               # 配置文件
├── tests/                # 测试文件
├── requirements.txt      # 依赖列表
└── build.spec           # 打包配置
```

### 技术栈

- **GUI 框架**: PyQt5
- **系统 API**: Windows GDI32
- **热键库**: keyboard
- **配置格式**: JSON
- **打包工具**: PyInstaller

### 构建可执行文件

```bash
# 安装打包工具
pip install pyinstaller

# 打包程序
pyinstaller build.spec

# 输出文件在 dist/ 目录
```

## ❓ 常见问题

### Q: 程序需要管理员权限吗？
A: 不需要。程序使用标准的 Windows API，普通用户权限即可运行。

### Q: 设置会在重启后保留吗？
A: 会的。所有设置都会自动保存到配置文件，重启后自动恢复。

### Q: 支持多显示器吗？
A: 当前版本会同时调节所有显示器。多显示器独立调节功能计划在后续版本中添加。

### Q: 程序退出后屏幕会恢复正常吗？
A: 会的。程序退出时会自动恢复到系统默认设置。

### Q: 热键冲突怎么办？
A: 可以在设置中自定义热键，避免与其他程序冲突。
